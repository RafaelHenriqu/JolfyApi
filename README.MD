## 📚 Sobre o projeto

A JolfyApi foi uma API simples criada com o objetivo de fornecer conteúdos como:

* Piadas
* Nomes de séries
* Curiosidades
* Dicas gerais

Este projeto foi desenvolvido no início da minha jornada na programação, com a ideia de criar algo funcional e, ao mesmo tempo, exploratório — um projeto “real”, mesmo que simples.

Na época, também existia uma motivação pessoal de construir algo que pudesse, de alguma forma, se tornar útil ou até gerar valor no futuro.

---

## 🧠 Contexto de evolução

A primeira versão do projeto armazenava todos os dados em um arquivo JavaScript, o que limitava bastante a organização e escalabilidade.

Na atualização seguinte, houve uma melhoria estrutural importante:

* Migração dos dados para um arquivo JSON
* Melhor separação entre dados e lógica da aplicação
* Organização mais simples e acessível para uso

Hoje, esse tipo de estrutura é básica, mas na época representou um avanço importante no aprendizado.

---

## ⚙️ Endpoints (estrutura da API)

A API possui os seguintes endpoints:

* `GET /Piadas` → Retorna uma piada
* `GET /Series` → Retorna nomes de séries
* `GET /Curiosidades` → Retorna curiosidades
* `GET /Dicas` → Retorna dicas gerais
